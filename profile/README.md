# Rustzen

Rustzen maintains open-source components, engineering templates, and practical
tools for developer workflows.

The public organization page focuses on repositories that are inspectable and
maintained. Standalone app products are documented on the Rustzen website.

## Public Repositories

- [rustzen/rustzen-hub](https://github.com/rustzen/rustzen-hub) - Rustzen website, public docs, and product pages.
- [rustzen/rustzen-core](https://github.com/rustzen/rustzen-core) - shared Rust components and reusable crates for Rustzen projects.
- [rustzen/rustzen-admin](https://github.com/rustzen/rustzen-admin) - Web/Rust admin engineering template with backend, frontend, shared crates, and deployment assets.
- [rustzen/rustzen-zipper](https://github.com/rustzen/rustzen-zipper) - release packaging utility for repeatable zip archives in scripts and release workflows.
